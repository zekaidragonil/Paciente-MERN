

# Proyecto MERN - Gestión de Pacientes Veterinarios

Este repositorio es una práctica de desarrollo fullstack utilizando el stack MERN (MongoDB, Express, React, Node.js). Permite la gestión de pacientes veterinarios, autenticación de usuarios y administración de citas.

## Módulos del Proyecto
- **Backend**: API REST con Node.js, Express y MongoDB. Maneja autenticación (JWT), registro de usuarios, gestión de pacientes y recuperación de contraseñas.
- **Frontend**: Aplicación React con Vite y Tailwind CSS. Permite a los veterinarios gestionar pacientes, iniciar sesión y recuperar contraseñas.

## Funcionalidades principales
- Registro e inicio de sesión de veterinarios
- Autenticación y autorización con JWT
- CRUD de pacientes (crear, leer, actualizar, eliminar)
- Recuperación y cambio de contraseña por email

## Instalación y ejecución
1. Clona el repositorio:
	```bash
	git clone <url-del-repositorio>
	```
2. Instala dependencias y ejecuta el backend:
	```bash
	cd backend
	npm install
	npm run dev
	```
3. Instala dependencias y ejecuta el frontend:
	```bash
	cd ../Frontend
	npm install
	npm run dev
	```

## Notas
- Asegúrate de tener MongoDB en ejecución y configura las variables de entorno en el backend.
- Este proyecto es solo para fines educativos y de práctica.
