
# Frontend - Paciente MERN

Este proyecto es la parte frontend de una aplicación de gestión de pacientes veterinarios, desarrollado como práctica de aprendizaje con el stack MERN (MongoDB, Express, React, Node.js).

## Características
- Registro e inicio de sesión de veterinarios
- Gestión de pacientes (crear, editar, eliminar)
- Recuperación y cambio de contraseña
- Autenticación con JWT
- Interfaz moderna construida con React, Vite y Tailwind CSS

## Instalación y uso
1. Clona el repositorio y navega a la carpeta `Frontend`:
	```bash
	cd Frontend
	```
2. Instala las dependencias:
	```bash
	npm install
	```
3. Inicia la aplicación en modo desarrollo:
	```bash
	npm run dev
	```
4. Abre [http://localhost:5173](http://localhost:5173) en tu navegador.

## Notas
- Asegúrate de tener el backend corriendo para que la autenticación y la gestión de pacientes funcionen correctamente.
- Este proyecto es solo para fines educativos y de práctica.
